# Roles and Access

TPW uses its own access model and does not rely on WordPress roles for member permissions.

- Imported members are created with no WordPress role. This is intentional.
- Members with no role can still log in and use front‑end TPW screens.
- Front‑end access is controlled by a member’s status in TPW (e.g., Active, Honorary, Life Member).
- Avoid assigning default WP roles (like Subscriber) unless you want them to access the WordPress dashboard.

Admin tip: Use the Status filter and Advanced Search to audit membership states and keep your directory clean.
