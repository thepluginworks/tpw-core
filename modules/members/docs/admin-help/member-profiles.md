# Member Profiles and Self‑Service

Let members view and edit selected fields of their own profile.

## Configure editable and viewable fields

- Go to Member Settings → Member Profile.
- For each field, choose:
  - Editable by Member
  - Viewable by Member
- Some system fields are protected (not editable) or system‑only (not viewable).

## Select the Profile page

- Create a page in WordPress that contains the shortcode: '[tpw_member_profile]'
- In Member Settings → Member Profile, select this page in the dropdown.

## What members see

- A simple profile screen showing viewable fields.
- Inputs for fields you marked as editable.
- Save changes updates their member record immediately.

## Tips

- Keep editable fields to essentials to avoid data drift.
- Use Advanced Search in the directory to find members missing key data.
