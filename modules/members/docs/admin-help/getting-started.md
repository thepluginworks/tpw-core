# Getting Started for Admins

This quick guide helps you get set up and comfortable managing members.

## 1) Open the Members area

- Visit /manage-members/ while logged in as an admin.
- Use the search, filters, and view toggle (List or Card) to browse.

## 2) Add your first member

- Click Add New Member.
- Fill in the core fields (First Name, Surname, Email, Status, etc.).
- Save. You can edit later to upload a photo if enabled.

Tip: You can also Create from WP User to link an existing WordPress user to a TPW Member entry.

## 3) Import a CSV

- From the Members page, choose Tools → Import CSV.
- Download a sample from your developer if needed and match columns to TPW fields.
- After import, review a few members to verify data.

## 4) Basic settings

Go to Member Settings (top right) → General to configure:

- Allow admins to delete members
- Use Photos of Members (enables photos in the directory cards)
- Default Member Status for new entries

## 5) Directory visibility

Use Directory Field Visibility to choose which fields are shown to regular members in the directory.

## 6) Profile self‑service

Under Member Profile:

- Choose which fields a member can edit themselves
- Choose which fields a member can view on their own profile
- Select the page containing the [tpw_member_profile] shortcode

## 7) Address lookup

Under Address Lookup:

- Pick None, Ideal Postcodes, or Fetchify
- Enter provider credentials where required
- Use Test Lookup to verify a live provider when available

Next steps:
- Configure searchable fields (advanced search) in Field Settings.
- Encourage members to keep their profiles up to date.
