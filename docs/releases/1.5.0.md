## TPW Core v1.5.0 – Gallery Pagination and Elementor Widget

### Highlights
- Gallery shortcode: optional pagination controls for large galleries in Grid/List views.
  - New attributes: `per_page` and `paginate` for `[tpw_gallery]`.
  - Pagination uses per-gallery query args so multiple galleries on one page can paginate independently.
- Elementor (optional): TPW Gallery widget (Grid/List/Story) when Elementor is installed and active.
- Story view: improved navigation performance (preloads neighbor images) and `decoding="async"` on the main slide image.
- Public UI: fixed-column rendering support via a CSS variable for more predictable grids.

### How to Use
- Pagination (shortcode):
  - Enable with a default page size:
    - `[tpw_gallery id="123" view="grid" paginate="1"]`
  - Or set an explicit limit:
    - `[tpw_gallery id="123" view="list" per_page="40"]`
- Elementor widget:
  - Add the **TPW Gallery** widget in Elementor and select a gallery.
  - Choose View (Grid/List/Story); enable pagination for Grid/List if needed.

### Upgrade Notes
- No database schema changes.
- Elementor integration is fully guarded and only loads when Elementor is active.
- Cache keys now include pagination state to avoid serving the wrong gallery page.
