## TPW Core v1.5.2 – Gallery Heading Toggle & Caption Editor

- Gallery public display: added `show_heading` option (shortcode + Elementor) to hide the gallery title/description above images.
- Gallery Admin: caption editing now uses a textarea modal suitable for long narrative captions, with clamped caption previews so cards don’t expand.
