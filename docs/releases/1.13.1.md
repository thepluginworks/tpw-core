## TPW Core v1.13.1 - Noticeboard Admin Front-End Access

This release updates the active front-end noticeboard implementation so TPW members flagged as noticeboard admins can manage notices without requiring full WordPress administrator capability.

### Changed

- Active noticeboard management now allows access when either WordPress `manage_options` is granted or `TPW_Control_UI::is_noticeboard_admin()` returns true.
- The active notices shortcode render path and the AJAX save, delete, duplicate, and add-category handlers now share the same permission check to keep UI controls and server-side enforcement aligned.
- Existing WordPress administrator access remains unchanged.

See `CHANGELOG.md` under [1.13.1] - 2026-03-10 for full details.
