## TPW Core v1.13.2 - Conditional Custom Member Field Loader

This release adds a focused member field loader helper for conditional field integrations, making it easier to retrieve only the enabled custom checkbox fields that are approved for conditional logic.

### Added

- Added `TPW_Member_Field_Loader::get_condition_eligible_custom_fields()`.

### Changed

- The helper filters out core fields and non-checkbox fields before returning condition-eligible custom fields.
- Returned field metadata is normalized to sanitized `key`, `label`, and `type` values so consuming code can build conditional UI from a single trusted source.

See `CHANGELOG.md` under [1.13.2] - 2026-03-10 for full details.
