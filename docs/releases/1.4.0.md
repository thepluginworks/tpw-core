## TPW Core v1.4.0 – Gallery Help, UI Improvements, and Reordering

### Highlights
- New front-end Help page for the Gallery module at `/gallery-help/`.
- Shortcode `[tpw_gallery_help]` renders the help content on the system page.
- Admin: drag-and-drop image reordering for fast gallery organization.
- Image editing UI: streamlined caption editor and focal point controls.
- Performance: optimized styles and scripts for smoother interactions.
- Templates: refactored gallery templates for clearer structure and easier maintenance.
- Documentation: expanded Gallery Admin guide covering image management and categories.

### How to Use
- Help page: visit `/gallery-help/` or add `[tpw_gallery_help]` to any page.
- Reordering: use the Gallery Admin interface to drag thumbnails; order saves automatically.
- Captions & Focal Points: edit captions inline; adjust focal points in the editor for better crops.

### Upgrade Notes
- No database schema changes.
- The help page is automatically created by System Pages when the Gallery module is enabled.
