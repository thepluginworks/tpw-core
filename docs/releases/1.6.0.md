## TPW Core v1.6.0 – Members Households & DOB

This release expands the Members module with opt‑in Household support and adds a new core Date of Birth (DOB) field.

### Highlights

- Households (optional, default off): new admin tools on the Edit Member screen to create a household and assign/move members.
- Members: new core field Date of Birth (DOB).
- Field configuration: select fields can now define option lists via the `field_options` column (one option per line).

### Database / upgrades

- Members DB schema upgrades include:
  - `dob` column added to the members table.
  - New household tables for households and household membership.
  - `field_options` column added to field settings.
- Internal members DB version bumped to 0.3.6.

See `CHANGELOG.md` under [1.6.0] – 2026-01-13 for full details.
