## TPW Core v1.11.1 - Scheduler Diagnostics Patch

This release hardens the Core scheduler wrapper with better diagnostics around single-action scheduling.

### Added

- Request-scoped diagnostics helpers: `TPW_Core_Scheduler::get_last_error()`, `get_last_schedule_debug()`, and `get_schedule_debug_history()`.

### Changed

- `TPW_Core_Scheduler::schedule_single()` now stores richer context for both successful and failed scheduling attempts.
- Unique single scheduling now checks for existing hook/args/group matches before issuing a duplicate request to Action Scheduler.

See `CHANGELOG.md` under [1.11.1] - 2026-03-09 for full details.
