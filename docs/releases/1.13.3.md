## TPW Core v1.13.3 - Gallery Admin Member Flag

This release introduces a new core member flag for delegated gallery administration and tightens the active gallery admin access path so only approved users can reach the gallery management UI and endpoints.

### Added

- Added the core member field `is_gallery_admin` with the label `Gallery Admin`.

### Changed

- `is_gallery_admin` now follows the same Core checkbox-style handling pattern as other permission-style member flags across schema upgrade, field settings, add/edit forms, profile protections, and the member details modal.
- The active front-end gallery admin shortcode and gallery management AJAX/template paths now allow only WordPress admins with `manage_options`, TPW members with `is_admin = 1`, or TPW members with `is_gallery_admin = 1`.
- Gallery admin access now resolves through a shared helper so rendered UI and action handlers stay aligned.

See `CHANGELOG.md` under [1.13.3] - 2026-03-10 for full details.
