## TPW Core v1.7.2 – Members Admin Form Validation

This release fixes Members admin form validation issues that affected imported member records.

### Fixes

- Edit Member: saving is no longer blocked when an imported member has an empty `tpw_members.username`.
- Add Member: required fields are validated inline (username, first name, surname, email) instead of failing after submit.

### Minor UX

- Edit Member: when the member username is blank but a linked WordPress user exists, the WordPress `user_login` is shown in the Username field (display-only; no syncing).

See `CHANGELOG.md` under [1.7.2] – 2026-01-23 for full details.
