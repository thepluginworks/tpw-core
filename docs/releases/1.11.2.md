## TPW Core v1.11.2 - Scheduler Wrapper Diagnostics

This release expands the scheduler wrapper diagnostics so support and debugging can tell whether Action Scheduler calls were intercepted, short-circuited, or returned unexpected values.

### Added

- New `TPW_Core_Scheduler::get_wrapper_diagnostics()` helper for reporting the loaded wrapper file, wrapper version context, Action Scheduler source/version, and registered `pre_as_schedule_single_action` callbacks.

### Changed

- `TPW_Core_Scheduler::schedule_single()` now records raw scheduler return values and branch-specific context for wrapper validation failures, pre-filter short circuits, duplicate detection, thrown exceptions, zero returns, and successful scheduling.
- Optional admin-only debug logging can now emit before/after call events when `TPW_CORE_SCHEDULER_DEBUG_LOG` is enabled.

See `CHANGELOG.md` under [1.11.2] - 2026-03-09 for full details.
