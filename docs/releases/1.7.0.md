## TPW Core v1.7.0 – Scheduler Engine (Action Scheduler)

This release adds a stable scheduling layer to TPW Core using the bundled Action Scheduler library.

### Highlights

- Core Scheduler Manager: `TPW_Core_Scheduler` is the single source of truth for scheduling across TPW plugins.
- Lazy initialization: Action Scheduler is only loaded when needed, and Core will use an existing Action Scheduler instance when another plugin (e.g. WooCommerce) already loaded it.
- Stable wrapper API: helpers to schedule single and recurring tasks, unschedule tasks, and query whether tasks exist. Default group is `tpw`.

### Developer note

Dependent TPW plugins should call `TPW_Core_Scheduler::init_if_needed()` early (e.g. on `plugins_loaded`) and should not bundle Action Scheduler.

See `CHANGELOG.md` under [1.7.0] – 2026-01-20 for full details.
