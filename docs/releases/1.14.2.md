## TPW Core v1.14.2 - Payment Methods Navigation Cleanup

This release streamlines how administrators reach Payment Methods in TPW Core. The Payment Methods item under FlexiEvent now opens the current TPW Core Settings screen directly, while older compatibility routing has been removed now that active TPW plugins already use the shared destination.

### What's improved

- The FlexiEvent Payment Methods submenu now opens the current Payment Methods tab in TPW Core Settings.
- TPW Core no longer carries the obsolete legacy admin route for Payment Methods.
- Shared Core payment settings links continue to send administrators to the current Payment Methods settings screen.

See `CHANGELOG.md` under [1.14.2] - 2026-03-13 for full details.
