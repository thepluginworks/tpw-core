## TPW Core v1.10.0 – Same-Page Gallery Browser

This release adds a same-page public gallery browser for both shortcode and Elementor use.

### Added

- New `tpw_gallery_index` shortcode for rendering an index of galleries using each gallery's first image as the cover thumbnail.
- New Gallery Index Elementor widget that uses the same shared PHP render flow as the shortcode.

### Changed

- Clicking a gallery card now switches the current page from the gallery index to a single selected gallery view.
- The selected gallery view reuses the existing gallery renderer and adds a Back to Galleries action that returns to the index state on the same page.
- Public gallery browser pages now enqueue `tpw-buttons.css` so TPW button styles are available for embedded controls.

See `CHANGELOG.md` under [1.10.0] – 2026-03-08 for full details.
