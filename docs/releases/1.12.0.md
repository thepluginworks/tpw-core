## TPW Core v1.12.0 - Pluggable Profile Tabs and Central Email Dispatch

This release makes the My Profile screen extensible for TPW add-on plugins and centralizes outbound email dispatch behind a shared throttling-aware sender.

### Added

- My Profile tabs now render from the `tpw_core_register_profile_sections` registry so add-on plugins can register custom front-end profile sections with their own callbacks.
- New `TPW_Email::dispatch_mail()` helper provides a shared outbound email dispatcher with throttling-aware slot reservation and centralized logging hooks.
- Developer documentation now includes the My Profile tab extension contract, callback shape, member object contract, and an example integration snippet for future TPW plugins.

### Changed

- Built-in Profile and Payments sections now use the same normalized registry, priority sorting, and active-section rendering flow on the member-facing My Profile screen.
- Feedback submission emails and member-facing plain-text notifications now route through the shared email dispatcher when available.

See `CHANGELOG.md` under [1.12.0] - 2026-03-10 for full details.
