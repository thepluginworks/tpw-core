## TPW Core v1.14.1 - Payments Settings Persistence

This release restores TPW Core Payments currency settings persistence inside FlexiEvent settings by registering the TPW-owned keys with FlexiEvent's new allowed-keys filter.

### Changed

- Restored persistence for `currency_symbol` and `currency_code` in the FlexiEvent Payments settings tab.
- Added those TPW Core Payments keys to the new `flexievent_settings_allowed_keys` filter in the existing Payments settings tab file.

See `CHANGELOG.md` under [1.14.1] - 2026-03-13 for full details.
